Recreating the Team Alpha Male home page

The objective of this project was to recreate the framework of an already existing websites home page. I chose to work with the Team Alpha Male webpage for this assignment. To the left you'll see a thumbnail of the mockup I created, which is basically the websites blueprint. If you click on it you will be able to further disect the layout and placement of elements. This phase in the design process determines where all the content and information is going to be presented.


If you click on the thumbnail to the right you will be able to view the fully coded home page. Utilizing the mockup I created on moqups.com, I was able to code the layout of the page using HTML and CSS. Once I had all the frames in place, I was then able to apply all the content and images where they belonged in order to have the complete design of the page. For the purpose of this assignment we were adviced to have the page be 955 pixels in width and centered.

Responsive Web Development

The objective of this project was to create a make believe startup landing page with web responsive capabilities. HTML and CSS was used throughout the creation of the page as well as introductory bootstrap styling techniques. Some of the bootstrap classes that were applied on the page was the carousel in the testimonials section and the cards class in the pricing area. Web responsiveness was applied via the row and column bootstrap classes, which change the layout of page depending on the sizing of the browser or device being used.

HTML & CSS Exercises

HTML Elements

The purpose of this exercise was to gain familiarity with the use semantic elements. These types of elements are similar to divs but actually give meaning to different sections of the website code. With the use of semantic elements our code will be presented in a much more organized manner. Some of the elements utilized in this webpage were header, section, article, footer, nav, and aside.

Styling with Float

This assignments objective was to learn how to style images using float within our CSS. With the use of float we can have our websites images float to the left or to the right and have text flow around them as shown on this example.
